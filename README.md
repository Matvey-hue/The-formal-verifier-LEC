# The-formal-verifier-LEC
Разработка и реализация алгоритма формальной верификации на основе матричных вычислений.
- Здесь представлены результаты научно-исследовательской работы по задаче эквивалентности (Logic Equivalence Check, LEC).
- Реализация будет дорабатываться.
- Дата первой публикации исходного кода 07.04.2026
- На вход подаётся дискретная функция - на выход её Дизъюнктивная Нормальная Форма (ДНФ).
Если подать 2 функции, а между ними поставить XOR - на выход подастся ДНФ расхождения. В случае если выражения эквивалентны, ДНФ имеет ровно 0 импликант. Подробное описание смотреть в статье "Разработка и реализация алгоритма формальной верификации" (есть в репозитории).
- Структура репозитория:
- main.cpp - ядро алгоритма
- Parsrt AAG to TXT.cpp - утилита для перевода файла из формата AAG в TXT
- Разработка и реализация алгоритма формальной верификации.pdf - подробное описание научно-исследовательской работы: введение, математическое описание алгоритма, границы применимости, список использованной литературы
- Презентация.pptx - Презентация, с которой выступил на Международной Студенческой Научной Конференции (МСНК-79) в ГУАП (https://guap.ru/msnk). В течение 2 недель статья появится в сборнике конференции. Прикреплю сюда ссылку.
