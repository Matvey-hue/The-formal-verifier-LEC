# The-formal-verifier-LEC
Разработка и реализация алгоритма формальной верификации на основе матричных вычислений.
- Здесь представлен только исходный код на C++.
- Реализация будет дорабатываться.
- Дата первой публикации исходного кода 07.04.2026
- На вход подаётся дискретная функция - на выход её Дизъюнктивная Нормальная Форма (ДНФ).
Если подать 2 функции, а между ними поставить XOR - на выход подастся ДНФ расхождения. В случае если выражения эквивалентны, ДНФ имеет ровно 0 элементарных конъюнкций (термов).
* Полное описание алгоритма (идея, реализация, сложность, границы применимости) и результаты работы на бенчмарках будут опубликованы в сборнике 79-й Международной студенческой научной конференции ГУАП (https://guap.ru/msnk). В этот же день я там выступаю.

- Parsrt AAG to TXT.cpp - утилита для перевода файла из формата AAG в TXT
